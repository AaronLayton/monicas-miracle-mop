# Monica's Miracle Mop - Project Guide

## Overview
Professional house cleaning business website for Monica's Miracle Mop (UK-based). Built with Next.js App Router, Tailwind CSS v4, and shadcn/ui (base-nova style). The site enables customers to browse services, schedule appointments, and book cleaning jobs.

## Tech Stack
- **Framework:** Next.js 16 (App Router, Server Components by default)
- **Styling:** Tailwind CSS v4 with CSS variables via `@theme inline` in globals.css
- **Components:** shadcn/ui with base-nova style (uses @base-ui/react, NOT @radix-ui)
- **Icons:** lucide-react (NOT Material Symbols)
- **Font:** Plus Jakarta Sans via next/font/google
- **Language:** TypeScript
- **Package Manager:** npm

## Design System

### Colour Palette (from globals.css :root)
| Token | Hex | Usage |
|-------|-----|-------|
| --primary | #511983 (oklch purple) | Headings, primary actions |
| --primary-container | #6a359c | CTA buttons, featured cards |
| --primary-soft | #f0dbff | Light purple backgrounds |
| --secondary | #00687a (oklch teal) | Secondary actions, accents |
| --accent | #abedff | Light teal highlights |
| --background | #f8f9fe | Page background |
| --foreground | #191c1f | Body text |
| --muted | #f2f3f8 | Subtle backgrounds |
| --border | #cec3d3 | Borders and dividers |
| --destructive | #ba1a1a | Error states |

### Glass-Card Effects
Three utility classes in globals.css:
- `.glass-card` — blur(16px), 70% white bg — general use
- `.glass-card-nav` — blur(20px), 85% white bg — navigation bar
- `.glass-card-sidebar` — blur(12px), 70% white bg — booking summary sidebar

### Component Conventions
- UI primitives live in `components/ui/` (generated by shadcn CLI)
- Domain components live in feature directories: `components/booking/`, `components/schedule/`
- Shared layout components at `components/` root: nav.tsx, footer.tsx
- All components use `data-slot="component-name"` attribute
- Use `cn()` from `@/lib/utils` for className merging
- Use CVA for component variants
- Use composable sub-components (CardHeader/CardTitle/CardContent/CardFooter)
- Semantic colour classes only (bg-primary not bg-purple-500, never hardcoded hex)
- Use `gap-*` not `space-x-*` for flex spacing
- Import primitives from `@base-ui/react/[component]` (NOT @radix-ui)

### Server/Client Component Boundary
- Pages (app/*/page.tsx) are Server Components
- Interactive leaf components use "use client" (CalendarPicker, CounterInput, MobileNavToggle)
- Shared components (Nav, Footer, BookingSummary) are Server Components in Phase 1
