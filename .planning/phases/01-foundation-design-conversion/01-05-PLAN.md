1. Create `components/schedule/counter-input.tsx` with `"use client"`:

Props interface:
```tsx
interface CounterInputProps {
  label: string        // e.g. "Bedrooms"
  icon: React.ReactNode // Lucide icon element
  value: number
  onChange: (value: number) => void
  min?: number         // default 0
  max?: number         // default 10
  className?: string
}
```

- Use `data-slot="counter-input"` on root
- Display: icon + label on left, minus button + value + plus button on right
- Minus button: circular, `Button` variant="outline" size="icon-sm", Lucide `Minus` icon, disabled when value <= min
- Plus button: circular, `Button` variant="outline" size="icon-sm", Lucide `Plus` icon, disabled when value >= max
- Value display: `text-lg font-semibold` centered in a `w-10` fixed width
- Use `cn()` for className merging
- Export as named export

2. Create `components/schedule/calendar-picker.tsx` with `"use client"`:

Build a simple month grid calendar (do NOT use a date picker library -- per research "Don't Hand-Roll" guidance, build minimal version):

Props interface:
```tsx
interface CalendarPickerProps {
  selectedDate: Date | null
  onDateSelect: (date: Date) => void
  className?: string
}
```

- Use `data-slot="calendar-picker"` on root
- Header: month/year text (e.g. "April 2026") with `ChevronLeft` and `ChevronRight` navigation buttons
- Use `useState` for `currentMonth` (Date object tracking the displayed month)
- Day-of-week header row: Mo, Tu, We, Th, Fr, Sa, Su (UK week starts Monday)
- 6-row x 7-column grid of day cells
- Each cell: clickable button, shows day number
- Days from previous/next month shown greyed out (`text-muted-foreground`)
- Selected day: `bg-primary text-primary-foreground rounded-full`
- Today: `border border-primary` ring
- Past dates: slightly muted but still selectable (all dates available per D-11)
- Hoverable days: `hover:bg-muted rounded-full`
- Use `cn()` for conditional classes
- Export as named export

Phase 1: All dates appear available (no real availability data per D-11). State is local only (booking context comes in Phase 2).
